## 4) Como usar estas fuentes con IA (workflow sugerido)
Cuando se pida implementar algo en el proyecto:
1. Definir tarea concreta.
2. Consultar en este archivo la seccion relacionada (hosting, DNS o cloud).
3. Buscar 1 a 3 articulos oficiales antes de cambiar codigo o DNS.
4. Documentar en el PR/commit que articulo oficial se uso como base.
5. Aplicar cambios en entorno controlado y validar.

Plantilla de prompt sugerido para IA:
"Implementa [tarea]. Usa como referencia docs/Definicion de proyecto/donweb-centro-ayuda-fuentes.md y prioriza articulos de [hosting|DNS|cloud]. Antes de proponer cambios, lista riesgos de compatibilidad Donweb y validaciones necesarias."

## 6) CodeIgniter 4 en Donweb - Verificacion de compatibilidad

### Resultado: COMPATIBLE (verificado 2026-04-30)

Donweb no tiene articulo oficial dedicado a CodeIgniter 4. Sin embargo, la compatibilidad
fue verificada de forma indirecta via requisitos tecnicos:

| Requisito de CI4     | Estado en Donweb                        | Fuente                                   |
|---------------------|-----------------------------------------|------------------------------------------|
| PHP >= 8.1          | PHP hasta 8.4 disponible en panel Ferozo| Donweb CloudPanel / PHP version article  |
| Composer            | Disponible en cPanel/Ferozo             | Guia oficial Laravel en cPanel de Donweb |
| mod_rewrite         | Habilitado por defecto en hosting Linux | Estandar en planes Linux Donweb          |
| Escritura en disco  | Disponible (storage/, logs/)            | Estandar en hosting compartido           |
| MySQL/MariaDB       | Disponible via panel Ferozo             | Categoria Base de datos de Donweb        |

### Evidencia directa de soporte de frameworks PHP via Composer

Donweb tiene guia oficial para instalar Laravel en cPanel:
- https://guias.donweb.com/como-instalar-laravel-en-cpanel/

CI4 usa EXACTAMENTE el mismo mecanismo: `composer create-project codeigniter4/appstarter`.
Si Laravel funciona en Donweb via Composer, CI4 tambien funciona. Son equivalentes en requisitos.

### Links de referencia oficial CI4

- Documentacion oficial: https://codeigniter.com/user_guide/
- Requisitos de sistema: https://codeigniter.com/user_guide/intro/requirements.html
- Instalacion via Composer: https://codeigniter.com/user_guide/installation/installing_composer.html

### Notas para despliegue en Donweb

1. Subir via FTP solo la carpeta `public/` como raiz del hosting (o usar `.htaccess` para redirigir).
2. La carpeta `app/`, `system/` y `vendor/` deben quedar FUERA de `public_html` si el plan lo permite.
3. Si Donweb obliga todo dentro de `public_html`: mover `index.php` y `.htaccess` a la raiz y actualizar paths.
4. Configurar version de PHP a 8.1+ desde panel Ferozo antes de instalar.
